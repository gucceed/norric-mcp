# Anthropic Connector Directory — Partnership Track

## Status: NOT YET ELIGIBLE

The Anthropic MCP connector directory is **not a registry submission** — it is a
partnership process gated on demonstrated production traction.

## What it requires

- Paying customer base (internal Anthropic threshold: not public, but estimated
  10+ paying customers based on comparable connector listings)
- Direct relationship with an Anthropic BD or partnerships contact
- Likely a joint announcement or case study

## What it is NOT

- A form submission
- A GitHub PR
- A self-serve listing

## Action

Revisit this when norric-mcp has **10+ paying users**.

At that point:
1. Email partnerships@anthropic.com (or use the contact form at anthropic.com/contact)
2. Reference the mcp.so and PulseMCP listings as proof of ecosystem presence
3. Propose a co-marketing angle (Sweden/Nordic market, EU AI Act provenance,
   government registry data)

## Do not add this to the automated submission system.

The `registry/submit.py` CLI intentionally omits this. It cannot be automated.
